HTML Quotes :

HTML quotes are used to put a short quotation on your website. To do so, you need to use HTML q tag and HTML blockquote tag.

< stands for the less-than sign
> stands for the greater-than sign
≤ stands for the less-than or equals sign
≥ stands for the greater-than or equals sign
“ stands for the left double quotation mark
” stands for the right double quotation mark

HTML Attributes :

HTML attributes are special words which provide additional information about the elements or attributes are the modifier of the HTML element.


Align: which can use to indicate the alignment of paragraph on the page.
Id: which can be used to uniquely identify any element within an HTML page.
Title: which gives a suggested title for the element.
Class: which use to associate an element with a style sheet, and specifies the class of element.
Style: which allows you to specify Cascading Style Sheet (CSS) rules within the element.
dir: which allows you to indicate to the browser about the direction in which the text should flow.
lang: which allows you to indicate the main language used in a document, but this attribute was kept in HTML only for backwards compatibility with earlier versions of HTML. This attribute has been replaced by the xml:lang attribute in new XHTML documents.
